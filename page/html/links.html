MDN creating hyperlinks

Anchor / links properties

The <a> anchor tag, along with the href attribute, create a hyperlink. Links are the backbone of the internet. The first web page first web page contained 25 links.

Links can be created by <a>, <area>, <form>, and <link>.

The href attribute

MDN creating hyperlinks

While not required, the href attribute is found in almost all <a> tags. Providing the address of the hyperlink is what turns the <a> into a link. 

                                        
// 1
<a href="https://machinelearningworkshop.com">Machine Learning Workshop</a>
// 2
<a href="#teachers">Our teachers</a>
// 3
<a href="https://machinelearningworkshop.com#teachers">MLW teachers</a>
// 4
<a href="mailto:hal9000@machinelearningworkshop.com">Email Hal</a>
// 5
<a href="tel:8005551212">Call Hal</a>
