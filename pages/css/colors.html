CSS: Colors

CSS Colors

RGB Model: Every color can be represented by a combination of RED, GREEN and BLUE.

Each of the 3 base colors can take a value between 0 and 255, which leads to 16.8 million different colors

Defining colors in CSS:

  • RGB/RGBA Notation

    1. Regular RGB model

      • rgb(0, 255, 255)

    2. RBG with transparency ("alpha")

      • rgba(0, 255, 255, 0.3)

  • HEXADECIMAL Notation

    1. Instead of using scale from 0 to 255, we go from 0 to ff (255 in hexadecimal numbers)

      • #00ffff

    2. Shorthand, when all colors are identical pairs

      • #0ff

    3. Also have transparency

      • HEX Transparency

  • HSL (hue, saturation, lightness — тон, насыщенность и яркость

    1. hsl(315, 100%, 60%)

    2. hsla(359, 99%, 50%, 0.5)
