Базовая настройка Git на Windows/WSL2

Установка Git

  • Windows

  • WSL2 Ubuntu 'sudo apt install git'

Настройка Git

  • git config --global user.name "Artem Usatyi"

  • git config --global user.email miezaruexpr@gmail.com

  • git config --global core.editor "code --wait"

  • git config --global credential.helper cache

    • запоминание пароля (что-бы не спрашивал каждый раз при отправке изменений по HTTPS URL)

  • git config --list --show-origin

    • посмотреть все установленные настройки и где они заданы

  • git config --global alias.lg1 "log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold green)(%ar)%C(reset) %C(white)%s%C(reset) %C(dim white)- %an%C(reset)%C(bold yellow)%d%C(reset)' --all"

  • git config --global alias.lg2 "log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold cyan)%aD%C(reset) %C(bold green)(%ar)%C(reset)%C(bold yellow)%d%C(reset)%n'' %C(white)%s%C(reset) %C(dim white)- %an%C(reset)' --all"
